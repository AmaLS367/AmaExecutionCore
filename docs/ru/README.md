# Документация AmaExecutionCore

В этом каталоге хранятся технические спецификации, архитектурные схемы и разбивка компонентов.

## Основная документация

* **[Понимание системы](understanding_the_system.md)**: Концептуальное руководство простым языком о том, как работает бот, идеально подходит для начинающих и для общего обзора.
* **[Руководство по конфигурации](configuration.md)**: Подробное объяснение всех системных настроек и параметров `.env`.
* **[Руководство по API](api.md)**: Конечные точки выполнения и безопасности, формы запросов/ответов и операционные заметки.
* **[Архитектура](architecture.md)**: Границы модулей, поток оркестрации, раннеры (runners) и владение жизненным циклом.
* **[Руководство по бэктестированию и воспроизведению](backtest.md)**: Входные данные для исторического воспроизведения, форма вывода, метрики и границы режимов.
* **[Руководство по базе данных](database.md)**: Торговые таблицы, состояние безопасности и заметки о сохранении жизненного цикла.

## Запланированные документы

Первоначально запланированные документы теперь реализованы и хранятся в этом каталоге.
